Swik's portfolio ✦
Personal portfolio website for Swikriti Neupane — Data & ML Engineer.
Live site: okayswik.github.io

Stack

React — UI
Vite — build tool & dev server
gh-pages — deployment to GitHub Pages


Running locally
bashnpm install
npm run dev
Open http://localhost:5173

Deploying
bashnpm run deploy
Builds the app and pushes to the gh-pages branch. Live in ~ 2 minutes.
